# Vibe Coding

我的 AI 驱动开发工作空间。这里是代码、设计、文档和学习笔记的集合地。

## 目录结构

```
.
├── src/               # 源代码
├── docs/              # 产品文档（PRD、需求、决策记录）
├── assets/
│   ├── design/        # 设计素材与效果图
│   ├── bug/           # 测试报错截图
│   └── reference/     # 参考图与灵感收集
├── notes/             # 学习笔记与踩坑记录
├── .claude/           # Claude Code 配置（不提交）
├── AGENTS.md          # AI 开发协作规范
└── README.md          # 本文件
```

## 快速开始

1. 需求文档从 `docs/` 开始
2. 代码写在 `src/` 里
3. 开发中遇到的问题记录在 `notes/`

## 协作方式

本项目由 **Claude Code** 辅助开发。所有 AI 协作规则详见 [AGENTS.md](AGENTS.md)。
